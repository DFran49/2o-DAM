API de la academia

Endpoints

  • http://localhost/academia/consultarTodos.php

    Devuelve un json con todos los cursos disponibles. Por cada curso se muestra su código identificador, su nombre, número de plazas totales, número de plazas ocupadas y el departamento que lo imparte. Este endpoint usa el método GET

  • http://localhost/academia/consultarCurso.php?codigo=código del curso

    Devuelve un json con la información del curso cuyo código se pasa en la url. Se devuelve el nombre del curso, su descripción y la url (relativa a la dirección base del servidor) de la foto. Usa el método GET

  • http://localhost/academia/matricular.php

    Sirve para matricular un alumno en un curso. Usa el método POST y necesita un json que tenga como claves: nombre (un String con el nombre del alumno), edad (un entero con la edad del alumno) y cod_curso (un String con el curso en el que se va a matricular). Actualmente este endpoint es experimental y no graba nada. Solo se limita de comprobar que los datos se han recibido correctamente e informar de ello
